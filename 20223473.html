关于HTML和CSS语言的一些简单介绍

HTML
全称Hyper Text Markup Language 中文含义：超文本标记语言
CSS
全称Cascading Style Sheets 中文含义： 层叠样式表

HTML页面的组织结构

  • html语言的目的是为了标记系统。
  • 第一点:通过document type来让浏览器知道是使用html5
  • 第二点: html根元素包括两个最大的元素 1.head元素 2.body元素
  • 第三点：body里面放内容
  • 第四点：html的结构很重要

css语言的目的

css语言是为了整理这个语言的外观

代码编译器

  • 代码编译器是为了提高效率
  • 重点表达结构，有利于我们理解
  • 代码高亮
  • 代码折叠，宏观理解代码

浏览器和web

  • 文件的概念需要记住，并且和电脑或者浏览器发生关系
  • 浏览器抓取文件再排版来制作一张网页
  • 代码优秀的性能，可以用很小的空间就可以控制互联网很多的资源
